# Emotion Recognition From Speech

This project was developed as part of the CodeAlpha Machine Learning Internship.

## Objective
Recognize human emotions from speech audio using Deep Learning techniques.

## Technologies Used
- Python
- TensorFlow
- Librosa
- NumPy
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

## Features
- Audio preprocessing
- MFCC feature extraction
- CNN-based emotion classification
- Accuracy visualization
- Confusion matrix analysis

## Dataset
RAVDESS Emotional Speech Audio Dataset

## Emotions Detected
- Happy
- Sad
- Angry
- Fearful
- Neutral
- Calm
- Disgust
- Surprised
